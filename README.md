# ObjetivoSquad
Objetivo Squad

Inicio: Calcular porcentaje del Squad

## Ingreso Datos
- punto_squad
- max_puntos
- porcentaje_squad

## Proceso
- Pedir punto_squad
- Pedir max_puntos
- Operar (100 * punto_squad) / max_puntos y guardar en porcentaje_squad
- Si porcentaje_squad es mayor que 80
	- Mostrar mensaje "Muy bien Squad!
- Si no
	- "Pueden hacerlo mejor"


*Fin*
